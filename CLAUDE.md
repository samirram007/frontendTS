# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
npm run dev          # Dev server on port 5173
npm run start        # Dev server on port 3000
npm run build        # Vite build + tsc
npm run test         # Vitest (all tests)
npm run test -- <pattern>   # Run tests matching a file/name pattern
npm run lint         # ESLint
npm run check        # Prettier + ESLint fix
```

> `src/routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin — never edit it manually.

## Environment

Copy `.env.example` to `.env`. Required variables:
- `VITE_API_BASE_URL` — Laravel backend base URL (e.g. `https://aipt-api.local/api`)
- `VITE_BASE_URL` — app base path for Vite
- `VITE_API_SECURE` — `true`/`false` for HTTPS proxy

Dev server proxies `/api/*` to the backend.
